1. Zero-shot Prompting

Use Case:

Tasks where no examples are provided, and the model must rely on its pre-trained knowledge.

Example:

Prompt: "Translate the following English text to French: 'Hello, how are you?'"

Output: "Bonjour, comment ça va?"

Developed By:

OpenAI (GPT models)

2. Few-shot Prompting

Use Case:

Tasks where a few examples are provided to guide the model.

Example:

Prompt: "Translate English to French: 'Hello' -> 'Bonjour', 'Goodbye' -> 'Au revoir', 'Thank you' -> ?"

Output: "Merci"

Developed By:

OpenAI (GPT models)

3. Chain-of-Thought Prompting

Use Case:

Complex reasoning tasks requiring step-by-step thinking.

Example:

Prompt: "If John has 5 apples and gives 2 to Mary, how many apples does he have left? Let's think step by step."

Output: "John starts with 5 apples. He gives 2 to Mary. 5 - 2 = 3. John has 3 apples left."

Developed By:

Google Research

4. Meta Prompting

Use Case:

Guiding the model to generate or refine other prompts.

Example:

Prompt: "Generate a prompt to summarize a long article."

Output: "Summarize the following article in 3 sentences: [Article Text]"

Developed By:

OpenAI

5. Self-Consistency

Use Case:

Improving reliability by generating multiple responses and selecting the most consistent one.

Example:

Prompt: "What is the capital of France?"

Output: "Paris" (selected from multiple consistent responses)

Developed By:

Google Research

6. Generate Knowledge Prompting

Use Case:

Tasks requiring factual accuracy by generating relevant knowledge first.

Example:

Prompt: "What is the boiling point of water? First, generate relevant knowledge."

Output: "The boiling point of water is 100°C at standard atmospheric pressure."

Developed By:

OpenAI

7. Prompt Chaining

Use Case:

Breaking down complex tasks into smaller, sequential prompts.

Example:

Prompt 1: "Summarize the following article: [Article Text]"

Prompt 2: "Based on the summary, generate a list of key points."

Developed By:

Research Community

8. Tree of Thoughts

Use Case:

Exploring multiple reasoning paths to solve complex problems.

Example:

Prompt: "Solve the following math problem by exploring multiple approaches: 2x + 3 = 7."

Output: "Approach 1: Subtract 3 from both sides. Approach 2: Divide both sides by 2."

Developed By:

Research Community

9. Retrieval Augmented Generation

Use Case:

Tasks requiring external knowledge retrieval combined with generation.

Example:

Prompt: "What is the population of France? Retrieve relevant data and generate an answer."

Output: "The population of France is approximately 67 million."

Developed By:

Facebook AI

10. Automatic Prompt Engineer

Use Case:

Automating the creation of effective prompts for specific tasks.

Example:

Prompt: "Generate a prompt to classify sentiment in a text."

Output: "Classify the sentiment of the following text as positive, negative, or neutral: [Text]"

Developed By:

OpenAI

11. Automatic Reasoning and Tool-use

Use Case:

Tasks requiring reasoning and integration with external tools.

Example:

Prompt: "Calculate the square root of 144 using a calculator tool."

Output: "The square root of 144 is 12."

Developed By:

OpenAI

12. Active-Prompt

Use Case:

Dynamically refining prompts based on model feedback.

Example:

Prompt: "Generate a summary of the following text: [Text]"

Output: "The text discusses the benefits of exercise for mental health."

Developed By:

Research Community

13. Directional Stimulus Prompting

Use Case:

Guiding the model's output using specific cues or hints.

Example:

Prompt: "Write a story about a dragon. Hint: The dragon is friendly and loves to bake."

Output: "Once upon a time, there was a friendly dragon named Spark who loved baking cookies."

Developed By:

Research Community

14. Program-Aided Language Models

Use Case:

Tasks requiring integration of programming logic with language models.

Example:

Prompt: "Write a Python function to calculate the factorial of a number."

Output: "def factorial(n): return 1 if n == 0 else n * factorial(n-1)"

Developed By:

OpenAI

15. ReAct

Use Case:

Combining reasoning and action for task completion.

Example:

Prompt: "Plan a trip to Paris. First, find flights, then book a hotel."

Output: "Step 1: Search for flights. Step 2: Book a hotel near the Eiffel Tower."

Developed By:

Google Research

16. Reflexion

Use Case:

Improving model outputs through iterative reflection and refinement.

Example:

Prompt: "Write a poem about nature. Reflect and improve the first draft."

Output: "The first draft: 'The trees are green.' Improved: 'The lush green trees sway in the gentle breeze.'"

Developed By:

Research Community

17. Multimodal CoT

Use Case:

Combining text and visual inputs for reasoning tasks.

Example:

Prompt: "Given an image of a cat, describe its features step by step."

Output: "Step 1: The cat has black fur. Step 2: It has green eyes. Step 3: It is sitting on a mat."

Developed By:

Research Community

18. Graph Prompting

Use Case:

Tasks involving structured data or relationships represented as graphs.

Example:

Prompt: "Given a graph of friends, who is the most connected person?"

Output: "Alice is the most connected person with 5 friends."

Developed By:

Research Community
